Workflow outline

A central repository is the communication hub for all developers. This guide assumes the central repository has been set up.

Client side setup

First download a Git Client. Recommended tools for Windows are

  • SourceTree http://www.sourcetreeapp.com/
  • Github for Windows https://windows.github.com/

Client side setup

SourceTree is strong in visually representating repositories and has built-in Git flow support.

GitHub for Windows has an excellent Windows native Powershell Git Shell

Open up the commandline terminal from SourceTree by clicking 'Terminal'

Client side setup

Configure name and email on first time setup. > git config --global user.name "Your Name" > git config --global user.email "your_email@whatever.com"

Setup line ending preferences. For Unix/Mac: > git config --global core.autocrlf input > git config --global core.safecrlf true For Windows: > git config --global core.autocrlf true > git config --global core.safecrlf true
